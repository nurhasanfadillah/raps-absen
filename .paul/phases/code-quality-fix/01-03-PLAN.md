## Goal
Pecah `store.tsx` (615 baris) menjadi modul-modul `store/` yang terpisah per domain, dan hapus `lib/db.ts` beserta dependency `idb` yang tidak digunakan.

## Purpose
`store.tsx` yang 615 baris menggabungkan auth, employee, attendance, kasbon, payroll, dan audit log dalam satu file — tidak bisa dinavigasi, tidak bisa ditest, dan sulit di-extend. `lib/db.ts` adalah dead code (IndexedDB schema terdefinisi tapi tidak pernah digunakan).

## Output
- `store/` directory dengan 7 file terpisah
- `store.tsx` dihapus, diganti `store/index.tsx` sebagai public entry point
- `lib/db.ts` dihapus
- `idb` diuninstall dari `package.json`
- Semua import `from '../store'` dan `from './store'` di pages tetap berfungsi (public API tidak berubah)
</objective>
