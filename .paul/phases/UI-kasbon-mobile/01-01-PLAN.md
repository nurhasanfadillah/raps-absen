<objective>
## Goal
Audit dan improve halaman Kasbon untuk mobile: tambah sort/filter/pagination, redesign card menjadi compact 2-kolom dengan expand on tap, dan hapus restriksi edit/hapus berdasarkan status.

## Purpose
Halaman kasbon saat ini verbose di mobile — setiap card menampilkan semua info sekaligus tanpa filter/sort. Pengguna perlu bisa koreksi data apapun statusnya (salah input, salah proses).

## Output
`pages/CashAdvance.tsx` diperbarui dengan semua fitur baru.
</objective>

<acceptance_criteria>

## AC-1: Sort dan filter berfungsi
```gherkin
Given halaman kasbon terbuka dengan data campuran
When user memilih filter "Pending" dan toggle sort ke "Terlama"
Then hanya record pending yang tampil, diurutkan dari tanggal terlama
  And currentPage reset ke 1 saat filter/sort berubah
```

## AC-2: Card compact 2-kolom dengan expand
```gherkin
Given daftar kasbon tampil dalam list compact
When user melihat card dalam collapsed state
Then card menampilkan: kiri-atas=tanggal(dd/mm), kiri-bawah=badge status, kanan-atas=nama(rata kiri), kanan-bawah=nominal(rata kanan)
  And tidak ada detail lain yang terlihat

When user tap/klik card
Then card expand menampilkan: alasan, tanggal penuh, dan action buttons (Tandai Lunas, Edit, Hapus)
  And tap ulang menutup kembali
```

## AC-3: Edit dan hapus tanpa restriksi status
```gherkin
Given kasbon dengan status "Lunas" atau "Potong Gaji"
When user expand card tersebut
Then tombol Edit dan Hapus tetap muncul dan bisa diklik
  And openEdit tidak menolak dengan toast error
  And handleDelete tidak menyebutkan "Data yang sudah lunas tidak dapat dihapus"
```

## AC-4: Pagination berfungsi
```gherkin
Given ada lebih dari 10 data kasbon
When halaman pertama kali dibuka
Then hanya 10 item pertama ditampilkan
  And paginator menampilkan nomor halaman dan tombol prev/next
  And ada selector page size dengan opsi 10/25/50

When user ganti page size ke 25
Then menampilkan hingga 25 item dan paginator diperbarui
```

  <name>Task 3: Sort/filter controls dan pagination UI</name>
  <files>pages/CashAdvance.tsx</files>
  <action>
    **A. Tambah controls bar** — sisipkan di antara header (div judul + tombol "Ajukan Kasbon") dan daftar card. Letakkan dalam `div` terpisah:

    ```tsx
    {/* Controls: Filter + Sort */}
    <div className="flex flex-col sm:flex-row gap-2 sm:items-center sm:justify-between">
      {/* Filter pills */}
      <div className="flex gap-1 bg-slate-900 border border-slate-800 rounded-lg p-1 w-fit">
        {(['all', 'pending', 'paid'] as const).map(val => (
          <button
            key={val}
            onClick={() => changeFilter(val)}
            className={`px-3 py-1.5 rounded-md text-xs font-medium transition-colors ${
              filterStatus === val
                ? 'bg-slate-700 text-slate-100'
                : 'text-slate-400 hover:text-slate-200'
            }`}
          >
            {val === 'all' ? 'Semua' : val === 'pending' ? 'Pending' : 'Lunas'}
          </button>
        ))}
      </div>

      {/* Sort toggle */}
      <button
        onClick={changeSort}
        className="flex items-center gap-1.5 text-xs text-slate-400 hover:text-slate-200 px-3 py-2 rounded-lg border border-slate-800 bg-slate-900 hover:bg-slate-800 transition-colors w-fit"
      >
        <ArrowUpDown size={13} />
        {sortOrder === 'desc' ? 'Terbaru' : 'Terlama'}
      </button>
    </div>
    ```

    **B. Tambah pagination footer** — sisipkan setelah daftar card (setelah closing `div` list), sebelum `Modal`:

    ```tsx
    {/* Pagination */}
    {sorted.length > 0 && (
      <div className="flex flex-col sm:flex-row items-center justify-between gap-3 pt-2 border-t border-slate-800">
        {/* Info + page size */}
        <div className="flex items-center gap-3 text-xs text-slate-500">
          <span>
            {((currentPage - 1) * pageSize) + 1}–{Math.min(currentPage * pageSize, sorted.length)} dari {sorted.length} data
          </span>
          <select
            value={pageSize}
            onChange={e => { setPageSize(Number(e.target.value)); setCurrentPage(1); }}
            className="bg-slate-900 border border-slate-700 rounded-md px-2 py-1 text-xs text-slate-300 focus:outline-none focus:border-slate-500"
          >
            {[10, 25, 50].map(n => (
              <option key={n} value={n}>{n} / halaman</option>
            ))}
          </select>
        </div>

        {/* Page buttons */}
        <div className="flex items-center gap-1">
          <button
            onClick={() => setCurrentPage(p => Math.max(1, p - 1))}
            disabled={currentPage === 1}
            className="h-8 w-8 flex items-center justify-center rounded-md border border-slate-700 bg-slate-900 text-slate-400 hover:text-slate-200 hover:bg-slate-800 disabled:opacity-30 disabled:cursor-not-allowed transition-colors"
          >
            <ChevronLeft size={14} />
          </button>

          {Array.from({ length: totalPages }, (_, i) => i + 1)
            .filter(p => p === 1 || p === totalPages || Math.abs(p - currentPage) <= 1)
            .reduce<(number | '...')[]>((acc, p, idx, arr) => {
              if (idx > 0 && typeof arr[idx - 1] === 'number' && (p as number) - (arr[idx - 1] as number) > 1) {
                acc.push('...');
              }
              acc.push(p);
              return acc;
            }, [])
            .map((p, idx) => p === '...' ? (
              <span key={`ellipsis-${idx}`} className="h-8 w-8 flex items-center justify-center text-xs text-slate-600">…</span>
            ) : (
              <button
                key={p}
                onClick={() => setCurrentPage(p as number)}
                className={`h-8 w-8 flex items-center justify-center rounded-md text-xs font-medium transition-colors ${
                  currentPage === p
                    ? 'bg-slate-700 text-slate-100 border border-slate-600'
                    : 'border border-slate-700 bg-slate-900 text-slate-400 hover:text-slate-200 hover:bg-slate-800'
                }`}
              >
                {p}
              </button>
            ))
          }

          <button
            onClick={() => setCurrentPage(p => Math.min(totalPages, p + 1))}
            disabled={currentPage === totalPages}
            className="h-8 w-8 flex items-center justify-center rounded-md border border-slate-700 bg-slate-900 text-slate-400 hover:text-slate-200 hover:bg-slate-800 disabled:opacity-30 disabled:cursor-not-allowed transition-colors"
          >
            <ChevronRight size={14} />
          </button>
        </div>
      </div>
    )}
    ```
